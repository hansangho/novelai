# 문체 가이드 (Style Bible)

> PRD v1.4 §4.2. Style Linter 가 이 파일의 규칙을 읽어 Gate G1 을 수행한다.
> **집필 단계 진입 후 Read-only**. 수정은 `/bible-unlock` 후에만.

## 서술 시점
- 3인칭 제한 시점을 기본으로 한다.
- 챕터 단위로 POV 인물을 고정한다. 교차가 필요하면 `---` 섹션 구분자로 분리한다.

## 시제
- 주 서술은 과거형을 원칙으로 한다.
- 회상·대과거는 `~했었다` 형태로 층위를 구분한다.

## 문장 리듬
- 평균 목표 길이 15~25자. 장면 태그에 따라 조정된다 (`부록 A`).
- 긴장·액션 장면: 짧게 (5~15자).
- 서정·묘사 장면: 길게 (20~40자).
- 같은 길이 문장이 4회 이상 연속되지 않도록.

## 어휘
### 금지 어휘 (Linter 기본 규칙)
- `그러나` → `그런데` / `하지만`
- `~의 것이다` (번역투)
- 이모티콘·구어 남용: `ㅎㅎ`, `...` 3회 이상 반복

### 선호 표현
- 감정을 행동으로: "슬펐다" → "창밖을 오래 바라봤다"
- 설명보다 구체적 이미지

## 금지 패턴 (AI slop 방지)
- 챕터 시작구: `그날 아침,`, `운명의 그날,`, `~는 생각했다.` 로 시작 금지
- 한 문장에 `매우/정말/너무/아주` 중 2개 이상 동시 등장 금지
- 같은 명사·동사를 3문장 안에서 3회 이상 반복 금지

## 대사 규칙
- 직접 인용부호는 `" "` 를 사용한다.
- 지시문은 대사 뒤 또는 분리된 줄에 붙인다.
- `"~라고 말했다"` 는 연속 2회 이상 금지. 동사 교체 (물었다 / 중얼거렸다 / 내뱉었다 …) 혹은 생략.

## 장면 태그 (scene override)
문장 상단 주석 `<!-- scene: action -->` / `<!-- scene: lyrical -->` / `<!-- scene: dialogue -->` 로 임계값을 다르게 적용한다. 부록 A 의 `scene_overrides` 참조.
