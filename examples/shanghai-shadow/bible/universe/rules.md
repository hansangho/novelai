# 세계의 규칙 (상하이의 그림자)

## 물리·시공간
- 현실 기반. 비현실 요소 없음.

## 사회·제도 (1923년 조선총독부 체제)
- 법 체계: 일본 제국 법 + 조선 총독부령. 조선인에 대한 차별적 적용 존재.
- 정보 통제: 신문 검열 상시. 독립운동 관련 보도 전면 금지.
- 통신 수단 (**반드시 지킬 것**):
  - 전보 (공적·사적 모두)
  - 우편 (조선-상하이 최소 5일)
  - 대면
  - 비공식: 인편 쪽지
  - **휴대전화 / 무선통신 / 개인 전화 — 존재하지 않음**

## 무기 / 폭력 묘사
- 경찰 권총: 32구경. 장전 5발.
- 구타 묘사는 가능하나 고문은 서사상 필요할 때만 간접 묘사.
- 폭력은 결과(부상·트라우마)가 이후 챕터까지 이어져야 한다 — `state/chapter-NN/character-states.yaml#injuries` 에 기록.

## 금기 (절대 위반 금지)
1. 시대착오 (현대 기술, 현대 어휘)
2. 조선인 캐릭터가 일본인 이름을 (작중) 사적 자리에서 사용
3. 종교·민족 고정관념의 무비판적 사용
