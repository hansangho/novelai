# 플롯 아키타입 (작가 프로젝트용 빠른 참조)

> 30 가지 플롯 아키타입의 압축 버전. 풀 카탈로그는 플러그인 저장소의 `docs/PLOT-ARCHETYPES.md`.
> Structure Architect 가 구조 설계 시 이 파일을 참조한다.
> 작가 본인이 작품의 핵심 아키타입을 선택해 `bible/structure.md` 의 "플롯 프레임워크" 절에 명시할 것.

## 사용 절차

1. **작품의 메인 아키타입** — 1 개 선택
2. **보조 아키타입** — 1~2 개 (결합 가이드 참고)
3. **변주 요소** — 같은 아키타입을 어떻게 차별화할지 (인물·시공간·결합 비율·결말 톤)
4. **빈출 함정 인지** — 풀 카탈로그에서 해당 아키타입의 함정 확인
5. `bible/structure.md` 의 "플롯 프레임워크" 절에 메인 + 보조 + 이유 기록
