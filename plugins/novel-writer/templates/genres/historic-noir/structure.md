# 서사 구조 (역사 느와르 프리셋)

## 장르·톤
- 장르: 역사 느와르 / 첩보 스릴러 / 시대극 누아르
- 분위기: 차갑고 조여드는 긴장. 행간의 침묵이 대사보다 많다.

## 플롯 프레임워크 (권장)
- 3막 + 중간점 반전
- 또는 7-Point Story Structure (Hook → Plot Turn 1 → Pinch 1 → Midpoint → Pinch 2 → Plot Turn 2 → Resolution)

## 챕터 블록 (장편 18~25 챕터 표준)
| 블록 | 챕터 | 기능 |
|------|-------|------|
| Setup | 1~3 | 주인공 일상·세계 도입 + Inciting incident |
| Rising | 4~9 | 사건 추적·관계 구축 + 1st turning point (5~6장) |
| Midpoint | 10~12 | 진실 파편 노출 + Midpoint reversal |
| Descent | 13~17 | 주인공 도덕적 타협·고립 |
| Climax | 18~22 | 진실 충돌·선택·대가 |
| Resolution | 23~25 | 여진·재정렬 |

## 주요 서브플롯 (역사 느와르 표준)
- **SP-A 음모의 진짜 정체** — 1장 도입 / Midpoint reveal / 종장 회수
- **SP-B 동료의 정체** — 2~3장 도입 / 종반 reveal
- **SP-C 가족·과거의 그림자** — 1장 복선 / 중반 직면

## 금지선 (장르 표준)
- 데우스엑스마키나 결말
- 모든 악역이 단순한 악으로 환원
- 시대 고증 무시 (특히 통신 수단·이동 시간·복식)

## 추천 레퍼런스
- 김훈 『남한산성』, 『칼의 노래』 — 절제된 톤
- 하라 료 『그리고 밤은 되살아난다』 — 일본 하드보일드
- 레이먼드 챈들러 — 클래식 느와르 1인칭 감각
- 영화: 『암살』, 『밀정』, 『전우치』 (시대물 톤)
