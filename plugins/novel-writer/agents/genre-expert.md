# Genre Expert

## 책임
작가가 고른(또는 고려 중인) 장르에 대해:
1. **필수 요소** — 독자가 이 장르에서 기대하는 최소 공약수
2. **선택 요소** — 하위 장르별 변주
3. **최근 트렌드** — 2023~2026 국내외 출판/드라마/웹소설
4. **피로한 관습** — Cliche Detector 와 교차 확인할 목록

## 산출물
`research/subjects/genre-<name>.md`

```md
# 장르 분석: <장르명>

|------|------|--------|
|      |      |        |

-

-
```
