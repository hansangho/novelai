### 3.2 영구 미공개 lock
- **SP-A**: 박 부장 정확한 의도 / 박 부장 응답 / 1921 봉투 안 내용 / 박 부장 다음 행동
- **SP-B**: 이세아 본명 / 동지 사망 경위·이름 / 의열단 직속 라인 구체
- **SP-C**: 1921 사건 정체 / 뇌물 / 우편함 봉투 (수민 측 미접근)
- **SP-F**: 이세아 측 시선 / 관계 평가 / 도시·다방 이름·작전 단어

→ closure 원칙: **"모든 것을 한 번에 풀지 않는다"** (ch12 L.43 본문 명시).

### 3.3 신체 표지 closure 형식 — 결함 보존
- 1918 흉터: 같은 자리에 같은 모양으로 보존 (제거 아님)
- 시큰임: closure (오던 자리에 다른 감각) — ch12 이후 등장 0회
- 발소리 모티프: 도현 측 "자기 주인" / 이세아 측 "부재 보존" — 비대칭 거울 짝
- 결함 closure 어법: "L.95 그렇게는 살지 않게 되었네" — 자기 안의 한 줄, 그것에도 이름 안 붙임

→ 캐릭터 결함을 작품 종결에서 **없애지 않고** "보는 사람의 자리만 옮기는" 형식이 closure 다층 처리에 효과적.

### 3.4 닫지 않은 종결 표지 3행
ch12 마지막 카메라 = 종결 ≠ 봉인. 세 표지 잔존:
- 사립문 한 뼘 열린 채
- 흉터 같은 자리 같은 모양
- 서랍 안 봉투 그대로

→ 작품 종결 ≠ 모든 모티프 정리. **잔존 표지 3행** 이 종결 톤의 핵심 형식.
